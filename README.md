# Auraly

**Description**: Personalized daily horoscope notifications using scheduled tasks; tested with Postman APIs.

**Tech Stack**:
![Java](https://img.shields.io/badge/Java-ED8B00?style=for-the-badge&logo=java&logoColor=white)
![Spring Boot](https://img.shields.io/badge/SpringBoot-6DB33F?style=for-the-badge&logo=spring&logoColor=white)
![PostgreSQL](https://img.shields.io/badge/PostgreSQL-316192?style=for-the-badge&logo=postgresql&logoColor=white)
![Git](https://img.shields.io/badge/Git-F05032?style=for-the-badge&logo=git&logoColor=white)
![Maven](https://img.shields.io/badge/Maven-C71A36?style=for-the-badge&logo=apachemaven&logoColor=white)

**Setup Instructions**:
1. Clone repo: `git clone https://github.com/tanriverdili/auraly.git`
2. Build: `mvn clean install`
3. Run: `mvn spring-boot:run`

**Environment Variables**:
- `DB_URL` – PostgreSQL database URL
- `DB_USERNAME` – postgres
- `DB_PASSWORD` – 2025

**Usage Examples**:
- Send GET request to `/horoscope/today?userId=1` to get today's horoscope

**API Endpoints**:
- GET `/horoscope/today` – fetch today's horoscope
- POST `/user/register` – register new user
